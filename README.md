# ✨ Codetree TILs — 알고리즘 학습 아카이브

이 레포지토리는 **Codetree에서 공부한 문제 풀이 기록이 자동으로 커밋되는 저장소**입니다.  
매일 날짜별 폴더가 생성되며, 각 폴더 내부에는 **해당 날짜에 해결한 문제 목록과 학습 요약(README)**이 자동으로 포함됩니다.

이 저장소는 직접 작성한 프로젝트 repo가 아니라,  
📝 **알고리즘 학습 과정을 장기적으로 추적하기 위한 TIL(Time In Learning) 저장소**입니다.

---

## 🗂️ Repository Structure

codetree-TILs/  
├─ 20251126/  
│ ├─ 문제1/  
│ ├─ 문제2/  
│ └─ README.md ← 특정 날짜 학습 내용 자동 생성  
├─ 20251127/  
├─ 20251128/  
└─ ...

- 날짜(YYYYMMDD)별 폴더 자동 생성  
- 각 문제마다 자동 커밋  
- 날짜별 README에는 문제명·난이도·상태·코드 링크 등이 Codetree에서 자동 입력됨  

---

## 🎯 Learning Goal

Codetree의 Trail 커리큘럼을 기반으로,    
**기초 → 자료구조 → 알고리즘 → 고급 알고리즘**까지 단계적으로 완주하는 것을 목표로 합니다.
